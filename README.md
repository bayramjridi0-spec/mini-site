# Mini Site Blog

## Description
Ce projet consiste à développer un mini-site web statique pour apprendre l'utilisation de Git et GitHub dans un workflow collaboratif.  
Cette étape correspond à la **création de la homepage** avec HTML, CSS et JavaScript de base.

## Structure du projet
- `html/` : contient `index.html`  
- `css/` : contient `style.css`  
- `js/` : contient `script.js`  
- `README.md` : documentation du projet

## Fonctionnalités réalisées
- Page d’accueil (`index.html`) avec texte de bienvenue  
- Style de base appliqué via `style.css`  
- Script JavaScript simple dans `script.js` (alerte au clic sur un bouton)  

## Workflow Git utilisé
- Initialisation du dépôt local : `git init`  
- Branch principale : `main`  
- Commit initial : structure du projet  
- Feature branch : `feature-homepage`  
- Commits réalisés : 2  
  - `init: project structure`  
  - `feat: add basic homepage with HTML, CSS, JS`  
- Push sur GitHub : `git push -u origin main`
